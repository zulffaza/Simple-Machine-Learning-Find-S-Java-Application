Aplikasi implementasi algoritma Find-S Machine Learning dengan menggunakan bahasa java.

Cara kerja aplikasi :
1. Menginisialisasi array attribute sebagai dataset attribute, dataset attribute minimal memiliki 2 instance, 1 instance dan 1 lagi target
2. Mengecek apakah dataset tidak kosong
3. Mengecek apakah instance pada dataset lebih dari 1
4. Inisialisasi hipotesis awal dengan dataset awal yang memiliki target sesuai target yang diinginkan
5. Pengecekan instance, jika terdapat instance yang berbeda pada setiap dataset, maka instance tersebut akan diubah menjadi '*' / '?'
6. Menampilkan hipotesis yang didapatkan
